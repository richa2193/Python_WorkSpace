21. CREATE ADMIN USER VERIFIED BY DEFAULT
=================================================

Open:

models.py


Find this line:

verified = models.BooleanField(default=False)


Change it to:

verified = models.BooleanField(default=True)

-----------------------------------------
Why?
-----------------------------------------

Now every new admin user
will automatically become:

verified = True

and admin can login directly
without OTP verification.
