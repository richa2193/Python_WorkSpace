===============================
WHAT IS SOCIAL AUTHENTICATION?
===============================

Social Authentication means user can login using third-party services like:
	Google
	GitHub
	Facebook
	Twitter
Instead of creating separate username and password.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

=======================
INTERVIEW QUESTIONS
=======================

1. What is Social Authentication?
Social Authentication allows users to login using third-party providers like Google and GitHub.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

2. Which package is used for Social Authentication in Django?
Package:
pip install django-allauth

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

3. What is OAuth?
OAuth is an authorization protocol used for secure login using third-party providers.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

4. What is Client ID?
Client ID identifies your application to Google or GitHub.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

5. What is Secret Key?
Secret Key is a private security key used during authentication.
Never share Secret Key publicly.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------

6. What is Third Party API?
Third Party API means external services provided by another company.
Examples:
	Google API
	GitHub API
	Facebook API
