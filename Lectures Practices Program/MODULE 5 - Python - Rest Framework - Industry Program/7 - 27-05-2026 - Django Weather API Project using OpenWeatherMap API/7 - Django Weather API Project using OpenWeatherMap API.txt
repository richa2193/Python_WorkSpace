1. What is API?

API stands for:

Application Programming Interface

It allows communication between two applications.

Example: Django app ↔ Weather server

Interview Question
=========================

2. What is REST API?

REST API is used to:
•	Send data
•	Receive data
•	Communicate between applications

Methods:
•	GET → Fetch data
•	POST → Insert data
•	PUT → Update data
•	DELETE → Delete data
