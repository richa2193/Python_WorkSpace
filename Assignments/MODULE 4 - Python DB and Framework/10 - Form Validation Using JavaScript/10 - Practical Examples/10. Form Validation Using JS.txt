=================================================
DJANGO PROJECT USING JAVASCRIPT VALIDATION
FOR REGISTRATION FORM
=================================================

Validation Fields:

1. Email Validation
2. Phone Number Validation


=========================================
STEP 1 — CREATE DJANGO PROJECT
=========================================

Command:

django-admin startproject doctorfinder


=========================================
STEP 2 — OPEN PROJECT DIRECTORY
=========================================

Command:

cd doctorfinder


=========================================
STEP 3 — CREATE APP
=========================================

Command:

python manage.py startapp doctor


=========================================
STEP 4 — REGISTER APP
=========================================

Open:

doctorfinder/settings.py


Add:

'doctor',


inside:

INSTALLED_APPS


=========================================
STEP 5 — CREATE templates FOLDER
=========================================

Inside:

doctor

Create folder:

templates


=========================================
STEP 6 — CREATE HTML FILE
=========================================

Inside templates create:

register.html


Structure:

doctor/
    templates/
        register.html


=========================================
STEP 7 — WRITE HTML + JAVASCRIPT
=========================================

Open:

register.html


Write:

<!DOCTYPE html>
<html>

<head>

    <title>Registration Form</title>

    <script>

        function validateForm() {

            let email = document.forms["myForm"]["email"].value;

            let phone = document.forms["myForm"]["phone"].value;


            let emailPattern =
            /^[^ ]+@[^ ]+\.[a-z]{2,3}$/;


            let phonePattern =
            /^[0-9]{10}$/;


            if (!email.match(emailPattern)) {

                alert("Invalid Email");

                return false;
            }


            if (!phone.match(phonePattern)) {

                alert("Phone Number Must Be 10 Digits");

                return false;
            }


            alert("Form Submitted Successfully");

            return true;
        }

    </script>

</head>

<body>

    <h1>Doctor Finder Registration Form</h1>

    <form name="myForm"
          onsubmit="return validateForm()">

        <label>Name:</label>

        <input type="text" name="name" required>

        <br><br>


        <label>Email:</label>

        <input type="text" name="email" required>

        <br><br>


        <label>Phone:</label>

        <input type="text" name="phone" required>

        <br><br>


        <input type="submit" value="Register">

    </form>

</body>

</html>


=========================================
STEP 8 — CREATE VIEW
=========================================

Open:

doctor/views.py


Write:

from django.shortcuts import render


def register(request):

    return render(request,'register.html')


=========================================
STEP 9 — CREATE APP URL FILE
=========================================

Create:

doctor/urls.py


Write:

from django.urls import path
from . import views


urlpatterns = [

    path('',views.register),

]


=========================================
STEP 10 — CONNECT MAIN URL
=========================================

Open:

doctorfinder/urls.py


Write:

from django.contrib import admin
from django.urls import path,include


urlpatterns = [

    path('admin/', admin.site.urls),

    path('',include('doctor.urls')),

]


=========================================
STEP 11 — RUN SERVER
=========================================

Command:

python manage.py runserver


=========================================
STEP 12 — OPEN WEBSITE
=========================================

Open:

http://127.0.0.1:8000/


Registration form will appear.
