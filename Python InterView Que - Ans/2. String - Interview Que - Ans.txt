Que 1. 
What is a string in Python, and how do you create one? 
Ans.
A string is a sequence of characters (text).
Created using quotes. (‘ ’ or “ ”)
Ex:
name = "Richa"
Interview Line:
A string is a collection of characters used to store text data.


Que 2. 
Explain the immutability of strings in Python. What does it mean, and why is it important? 
Ans.
Strings are immutable – cannot be changed after creation.
Imp – you can create a new string, but not modify the original.
Ex:
s = "hello"
# s[0] = 'H'  error
Interview Line:
Immutability means strings cannot be changed, which makes them safe and memory efficient.

Que 3.

Ans.
Use indexing.
Ex:

print(s[0])  # P
interview Line:
we use index to access characters in a string.


Que 4.
What is string slicing, and how do you perform it in Python? 
Ans.
Used to get a part of string.
Syntax – string[start:end]
Ex:
s = "Python"
print(s[0:3])  # Pyt
interview Line:
slicing is used to extract a part of a string.
