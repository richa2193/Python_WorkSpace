Que 4.
Explain Python's dynamic typing and give an example. 
Ans.
Definition – python is a dynamically types language, which means you do not need to declare the data type of a variable explicitly.
The type is automatically decided at runtime based on the value assigned.
Ex:
x = 10        # x is an integer
print(type(x))

x = "Hello"   # now x becomes a string
print(type(x))
Output:
<class 'int'>
<class 'str'>

Que 6.
What is an if statement, and how is it used in Python?
Ans.
 An if statement is used to check a condition.
If the condition is true, the code inside the if block will run.
If the condition is false, the code will be skipped.
Ex:
age = 18

if age >= 18:
    print("You can vote")
Explanation:
Here, python checks the condition age >= 18. Since it is true, the message is printed.
Interview Line:
An if statement is used to check a condition and execute code only when the condition is true.

Que 7.
Explain the syntax of an if-elif-else structures.
Ans.
Used to check multiple conditions.
if – first condition.
elif – more conditions.
else – default case.
Ex:
marks = 75

if marks >= 90:
    print("A Grade")
elif marks >= 60:
    print("B Grade")
else:
    print("C Grade")
Interview Line:
It is used when we have multiple conditions, and only one block runs based on the condition.

Que 9.
How does the elif statement work in python? Provide an example.
Ans.
Elif means else if.
It checks another condition if previous one is false.
Ex:
num = 0

if num > 0:
    print("Positive")

    print("Zero")

Interview Line:

Que 10.
Explain the use of logical operators (and, or, not) in if statements.
Ans.
and – both conditions must be true.
or – at least one condition must be true.
not – reverse the condition.
Ex:
age = 20
has_id = True

if age >= 18 and has_id:
    print("Allowed")

if age < 18 or not has_id:
    print("Not allowed")
Interview Line:
Logical operators are used to combine multiple conditions in an if statement.

Que 15.
What is list comprehension, and how is it different from a for loop?
Ans.
List comprehension is a short and clean way to create lists.
Ex:
# for loop
nums = []
for i in range(5):
    nums.append(i)

# list comprehension
nums = [i for i in range(5)]
Interview Line:
List comprehension is a shorter and faster way to create lists compared to a for loop.

Que 19.
How can you use break and continue statements in a while loop?
Ans.
Break – Stops the loop.
Continue – Skips current iteration.
Ex:
i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    if i == 5:
        break
    print(i)
Interview Line:
Break exits the loop, continue skips one iteration.
